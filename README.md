This repository contains all the data analysis and figures that I have created for the paper: 

**"Biased Connectivity of Brain-wide Inputs to Ventral Subiculum Output Neurons"** by Ryan W.S. Wee and Andrew F. MacAskill. 

Link to paper: "https://www.sciencedirect.com/science/article/pii/S2211124720302679"

The datasets were cloned from a larger repository called **"OpenNeuroAI"** and contains some ideas on how the data could be manipulated and presented. 

**Required Python Libraries**:
1. Numpy
2. Pandas
3. Matplotlib
4. Plotly
